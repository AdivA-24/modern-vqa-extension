# Modern Cross-Lingual VQA

Compare ViLT (2021) vs LLaVA-1.6 (2024) for multilingual Visual Question Answering.

## Features

- **Multilingual Support**: Ask questions in 100+ languages
- **Cross-lingual Answers**: Get responses in any supported language
- **Modern VLMs**: Leverage latest vision-language models
- **Reasoning**: Understand why models give certain answers

## Models

- **Legacy**: ViLT-B32 (2021) - Lightweight vision-language transformer
- **Modern**: LLaVA-1.6-Mistral-7B (2024) - Advanced multimodal LLM
